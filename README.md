# a-share-multifactor-stock-selection
# A股多因子选股模型研究

## 项目简介
本项目基于A股全市场样本，构建包含动量、反转、规模和波动率的多因子选股模型，系统评估各因子的横截面预测能力，并验证多因子组合相较于单因子模型的优势。

研究遵循标准的量化金融工程流程，包括因子构建、数据预处理、单因子检验、多因子组合构建、分层回测以及因子贡献分析，最终形成具有投资价值的多因子选股策略。

---

## 研究目标
1. 验证动量、反转、规模和波动率因子在A股市场中的有效性。
2. 构建多因子综合评分模型，提升横截面收益预测能力。
3. 通过回归分析与逐步剔除法，评估各因子在组合中的边际贡献。
4. 构建多空组合，检验策略的投资可行性与风险收益特征。

---

## 因子构建

| 因子名称 | 计算方式 | 经济含义 |
|----------|----------|-----------|
| 动量因子（Momentum） | 过去20日收益率 | 趋势延续（在A股中呈现反转特征） |
| 反转因子（Reversal） | 过去5日收益率的相反数 | 短期均值回归 |
| 规模因子（Size） | 流通市值的对数 | 小市值溢价 |
| 波动率因子（Volatility） | 过去20日收益率标准差的相反数 | 低波动溢价 |

---

## 因子预处理流程

为保证因子结果的稳健性与可比性，本研究对所有因子进行了标准的工业化处理：

1. **去极值（Winsorize）**：采用5%和95%分位数截尾，减少极端值影响。
2. **标准化（Z-score）**：对横截面因子进行标准化处理。
3. **行业与市值中性化**：通过横截面回归剔除行业和规模暴露。
4. **再标准化**：保证中性化后因子的可比性。

---

## 单因子检验方法

- **Rank IC（Spearman相关系数）**
- **ICIR（IC均值 / IC标准差）**
- **IC正比例**
- **分层回测（Quantile Backtest）**
- **横截面回归分析**

### 单因子主要结论
- **波动率因子**：预测能力最强，是模型的核心因子。
- **规模因子**：具有稳定的补充信息。
- **反转因子**：在A股市场中表现有效。
- **动量因子**：呈现明显的短期反转特征，边际贡献较弱。

---

## 多因子模型构建

基于单因子 ICIR 的绝对值归一化结果，对各因子进行加权，构建多因子综合评分（Composite Factor），用于股票排序与组合构建。

### 多因子模型表现

| 指标 | 数值 |
|------|------|
| IC均值 | ≈ 0.103 |
| ICIR | ≈ 0.861 |
| IC正比例 | ≈ 80.7% |
| 多空组合年化收益 | ≈ 26.9% |
| Sharpe比率 | ≈ 1.47 |
| 最大回撤 | ≈ -13.6% |

多因子模型在预测能力和投资表现上均显著优于单因子模型。

---

## 因子贡献分析

为评估各因子在多因子模型中的作用，本研究采用以下方法：

1. **横截面回归分析**：比较各因子的平均回归系数与显著性。
2. **逐步剔除法（Leave-One-Out）**：分析剔除单个因子后模型性能的变化。
3. **ICIR权重分析**：评估因子在组合构建中的权重分布。

### 主要结论
- **波动率因子**：对模型贡献最大，是核心驱动因子。
- **规模因子**：提供重要补充信息。
- **反转因子**：具有辅助作用。
- **动量因子**：边际贡献最弱。

---

## 分层回测结果

多因子综合评分将股票划分为10个分组，高分组累计收益显著优于低分组，多空组合收益稳定向上，虽然分组收益未完全单调，但整体趋势清晰，符合真实市场特征。
